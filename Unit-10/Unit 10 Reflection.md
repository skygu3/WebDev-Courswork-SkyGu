# Unit 10 Reflection

[Reflect on your journey here]
In this unit, I learned how to bring HTML, CSS, and JavaScript together to build a complete multi-page website. I learned about file and folder structure, how to link external CSS and JavaScript files, how to build a navigation bar with the <nav> tag, and the basic principles of good web design.
One important thing I learned is how much organization matters. Keeping files in the right folders and using a consistent navigation bar across pages makes the site easier to maintain and gives users a better experience. I also learned that good design follows real principles like visual hierarchy, consistency, navigability, and color theory — it's not just about what looks "nice."
Overall, this unit tied everything from the course together. I now understand how a full website is built from start to finish, and I feel a lot more confident about designing and coding my own site.
