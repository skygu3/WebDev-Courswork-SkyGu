In this unit, I learned about the United Nations' 17 Sustainable Development Goals (SDGs) and how web design can be used to address real-world problems. I also got promoted from designing for one client to designing a multi-page website for a community need.
One important thing I learned is that good design isn't just about how a website looks — it's about whether it actually helps people. Picking a goal that matters in my own community made the project feel more meaningful, not just like another assignment.
Overall, this unit showed me that web development can be a tool for change. It also pushed me to think bigger and start planning a full multi-page site instead of a single page.
