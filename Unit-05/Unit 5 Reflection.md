# Unit 5 Reflection

[Reflect on your journey here]
In this unit, I learned about CSS and how to use it to style HTML elements. I learned the three ways to apply CSS, inline, internal, and external. Also basic properties like color, font-size, font-family, and background.
One important thing I learned is the idea of "cascading", as the name of CSS. When multiple styles apply to the same element, inline takes priority over internal and external. I also learned that listing fallback fonts is important in case a browser doesn't have the first one.
Overall, this unit showed me how CSS gives me creativity and control in web design, and how good styling improves the user experience.
