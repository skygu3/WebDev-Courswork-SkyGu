# Unit 8 Reflection

[Reflect on your journey here]
In this unit, I learned about CSS layout tools like the <div> tag, the class attribute, the Box Model, and the position and display properties. I also learned about responsive and adaptive design using media queries and the viewport meta tag.
One important thing I learned is how much control these properties give me over the layout of a web page. The Box Model helped me understand spacing with padding and margin, and display: flex made it easy to arrange elements horizontally without messy code. Media queries also taught me how to make a site look good on both phones and computers.
Overall, this unit was a big jump in what I could actually do with CSS. My pages went from looking like plain text on a screen to looking like real, organized websites.
