In this unit, I was introduced to JavaScript and how it adds interactivity to a website. I learned how to use functions like alert(), how to make buttons with the onClick event, and how to update HTML content using getElementById() and innerHTML.
One important thing I learned is that JavaScript is what makes a website feel alive. HTML gives structure and CSS adds style, but without JavaScript, the page just sits there. Even something simple like a pop-up or a button that changes text on click makes the site feel much more interactive.
Overall, this unit made me realize how all three languages, HTML, CSS, and JavaScript work together. I'm excited to use JavaScript to add more dynamic features to my websites in the future.
